# 🐹 打地鼠 - 微信小游戏

一个适合初学者学习的打地鼠微信小游戏，使用原生 JavaScript + Canvas 实现，无需任何外部框架。

![技术栈](https://img.shields.io/badge/技术栈-JavaScript%20Canvas-blue)
![难度](https://img.shields.io/badge/难度-初学者友好-green)

## 🎮 游戏玩法

1. 点击 **"开始游戏"** 按钮
2. 地鼠会从洞里随机冒出来
3. **快速点击地鼠** 来得分
4. 在 **30秒** 内尽可能获得高分！

### 地鼠类型

| 地鼠 | 颜色 | 分值 | 出现概率 |
|------|------|------|----------|
| 🟤 普通地鼠 | 棕色 | **10分** | 70% |
| 🔵 蓝地鼠 | 蓝色 | **20分** | 20% |
| 🟡 金地鼠 | 金色 | **30分** | 10%（速度更快）|

### 评分等级

- 🌟 **200分以上** — 地鼠杀手！
- 👍 **100分以上** — 反应神速！
- 😊 **50分以上** — 还不错哦~
- 💪 **50分以下** — 再接再厉！

## 🚀 如何运行

### 环境要求

- [微信开发者工具](https://developers.weixin.qq.com/miniprogram/dev/devtools/download.html)

### 步骤

1. **下载本项目**
   ```bash
   git clone https://github.com/chenby1988/---whack-a-mole.git
   ```

2. **导入项目**
   - 打开微信开发者工具
   - 点击 **"导入项目"**
   - 选择项目文件夹 `---whack-a-mole`
   - AppID 选择 **"测试号"**（或填入自己的小游戏 AppID）

3. **开始游戏**
   - 在左侧模拟器中点击 **"开始游戏"**
   - 用鼠标点击冒出的地鼠

## 📁 项目结构

```
---whack-a-mole/
├── game.js          # 游戏主逻辑（核心代码）
├── game.json        # 小游戏配置文件
├── project.config.json   # 微信开发者工具项目配置
└── README.md        # 项目说明文档
```

### 核心代码说明

| 模块 | 说明 |
|------|------|
| `CONFIG` | 游戏配置参数（时长、间隔、分值等） |
| `Hole` 类 | 绘制地洞，包含位置和状态 |
| `Mole` 类 | 地鼠的生成、动画、点击检测 |
| `Game` 类 | 游戏主循环、状态管理、UI渲染 |

## 🎨 技术特点

- **纯原生 JavaScript**，零依赖，代码简洁易懂
- **Canvas 绘制**，无需外部图片资源
- **完整的游戏循环**：菜单 → 游戏进行 → 结束评分
- **多种地鼠类型**：普通、蓝色、金色，增加趣味性
- **震动反馈**：打中地鼠时手机会轻微震动
- **动画效果**：地鼠上升/下降带缓动动画

## 🔧 自定义修改

打开 `game.js`，你可以轻松调整：

```javascript
const CONFIG = {
  GAME_TIME: 30,        // 游戏时长（秒）
  SPAWN_INTERVAL: 800,  // 地鼠出现间隔（毫秒，越小越快）
  MOLE_STAY_TIME: 1500, // 地鼠停留时间（毫秒）
  HIT_SCORE: 10,        // 基础分值
};
```
