# 🎮 打地鼠微信小游戏 - 新手教程

## 📁 文件说明

这个文件夹里有3个文件：

| 文件 | 作用 |
|------|------|
| `game.js` | **游戏主程序** - 包含所有游戏逻辑 |
| `game.json` | **游戏配置** - 微信小游戏的设置 |
| `project.config.json` | **项目配置** - 微信开发者工具用的 |

---

## 🚀 第一步：下载微信开发者工具

1. 打开电脑浏览器，访问：
   ```
   https://developers.weixin.qq.com/miniprogram/dev/devtools/download.html
   ```

2. 下载 **稳定版** Stable Build（根据你的电脑系统选择 Windows/Mac 版本）

3. 下载完成后，像安装普通软件一样安装它

---

## 🔧 第二步：导入项目

1. **打开** 微信开发者工具

2. 点击 **"导入项目"** 或 **"+"** 号

3. 点击 **"目录"** 那一行的选择按钮

4. 找到并选择这个文件夹：`whack-a-mole`
   > 就是包含 game.js 的那个文件夹！

5. **AppID** 那一栏：
   - 如果你有微信小程序账号，填你自己的 AppID
   - 如果没有，选择 **"测试号"** 或点击 **"小游戏"** 然后选游客模式

6. 点击 **"确定"**

---

## ▶️ 第三步：运行游戏

导入成功后，你会看到：

- **左侧**：模拟器（就像手机屏幕一样）
- **右侧**：代码编辑器

直接点击模拟器里的 **"开始游戏"** 按钮就能玩了！

操作方式：
- 🖱️ **鼠标点击** 地鼠（在电脑上测试时）
- 👆 **手指点击** 地鼠（在真机上）

---

## 🎯 游戏规则

| 地鼠颜色 | 分数 | 特点 |
|---------|------|------|
| 🟤 棕色 | 10分 | 最常见 |
| 🔵 蓝色 | 20分 | 出现较少 |
| 🟡 金色 | 30分 | 很少出现，速度更快！ |

- 游戏时间：**30秒**
- 点击地鼠即可得分
- 时间结束后显示最终得分

## 📱 第四步：在真机上体验

想在手机上玩？

1. 在微信开发者工具右上角，点击 **"真机调试"**
2. 用微信扫描弹出的二维码
3. 游戏就会在你的手机上运行！

## 🔍 常见问题

**Q: 提示 "game.json 不存在"？**
> A: 你选错了文件夹！要选择 `whack-a-mole` 这个文件夹，不是它的上级文件夹。

**Q: 屏幕是空白的？**
> A: 检查下面的 "Console" 控制台有没有红色报错。如果有，可能是代码有语法错误。

**Q: 怎么修改游戏时间？**
> A: 打开 `game.js`，找到这行：
> ```javascript
> GAME_TIME: 30,  // 把30改成你想要的秒数
> ```

**Q: 怎么让地鼠出现更快？**
> A: 打开 `game.js`，找到：
> ```javascript
> SPAWN_INTERVAL: 800,  // 数字越小，地鼠出现越快（单位：毫秒）
> ```

## 🎨 游戏画面说明

游戏使用 Canvas 绘制，不需要任何图片文件！所有的地鼠、洞、按钮都是用代码画出来的。

如果你想换颜色或样式，可以修改 `game.js` 里的颜色值，比如：
- `'#8B4513'` 是棕色
- `'#FFD700'` 是金色
- `'#FF6B6B'` 是红色

## 📚 想学习代码？

`game.js` 里有详细的注释，建议按这个顺序阅读：

1. `CONFIG` - 游戏配置参数
2. `Hole` 类 - 洞是怎么画的
3. `Mole` 类 - 地鼠的动画和绘制
4. `Game` 类 - 游戏主逻辑
